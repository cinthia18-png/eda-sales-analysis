## Context
This project performs an exploratory data analysis on Amazon sales data to better understand product performance, sales trends, and revenue distribution.

## Objective
Analyze sales data to identify top-performing products, sales trends over time, and key business insights.

## Dataset
- Source: Amazon Sales Report dataset
- Data type: E-commerce sales transactions
- Includes product details, order status, quantities, and sales values

## Tools
- Python (Pandas, NumPy, Matplotlib)
- Excel (initial data review)

## Process
1. Data loading and inspection
2. Data cleaning and preprocessing
3. Exploratory data analysis
4. Visualization of key patterns

## Key Findings
- A small number of products contribute to a significant portion of total sales.
- Sales performance varies across product categories.
- Clear sales patterns can be observed over time.

## Conclusions
The analysis provides insights that can help prioritize high-performing products and support data-driven business decisions.

## Repository Structure
- `data/`: dataset used for analysis
- `notebooks/`: Jupyter Notebook with full analysis
- `visuals/`: charts and visual outputs
